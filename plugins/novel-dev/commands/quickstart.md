# /quickstart

소설 완성까지 5단계로 안내하는 간편 워크플로우입니다.

## 사용법

```
/quickstart          -- 현재 상태에서 다음 단계 안내
/quickstart --reset  -- 처음부터 다시 시작
```

## 5단계

| Step | 명령 | 설명 |
|------|------|------|
| 1 | `/init` | 장르·분량·컨셉 입력 |
| 2 | `/design` | 세계관·캐릭터·플롯 자동 설계 |
| 3 | `/plot` | 회차별 플롯 생성 |
| 4 | `/write-all` | 자동 집필 |
| 5 | `/revise` | 퇴고 |

처음 사용자는 이 5단계만으로 소설을 완성할 수 있습니다.
고급 기능은 `/help --mode=expert`에서 확인하세요.
