[NOVEL-SISYPHUS: 메인 아크 설계]

## Idempotency

If output file already exists:
1. Read existing content
2. Preserve user modifications (fields not auto-generated)
3. Merge with new content
4. Warn if conflicting changes detected

## 실행 단계

1. **데이터 로드**
   - `plot/structure.json` (플롯 구조)
   - `characters/*.json` (주요 캐릭터)

2. **plot-architect 에이전트 호출**
   ```
   Task(subagent_type="plot-architect", prompt="
   프로젝트: {project.json}
   플롯 구조: {structure.json}
   캐릭터: {characters 요약}

   메인 아크를 **상세히** 설계해주세요:

   1. **핵심 갈등** (각 갈등당 300자 이상)
      - 외적 갈등: 환경/상황에서 오는 장애물, 구체적 상황
      - 내적 갈등: 주인공의 욕망 vs 두려움, 내면의 모순
      - 관계적 갈등: 인물 간 갈등의 원인과 전개

   2. **중심 질문 (Dramatic Question)**
      - 독자가 끝까지 알고 싶어하는 핵심 질문
      - 이 질문이 왜 독자의 관심을 끄는지 설명

   3. **주요 이벤트** (각 이벤트 설명 500자 이상)
      - Inciting Incident: 주인공의 일상이 어떻게 깨지는가
      - Plot Point 1: 1막→2막 전환, 돌아갈 수 없는 선택
      - Midpoint: 중간 반전, 게임 체인저 순간
      - Plot Point 2: 2막→3막, 가장 큰 위기
      - Climax: 최고조의 대결/결정
      - Resolution: 해결과 새로운 균형

   4. **주제/메시지**
      - 작품이 전달하고자 하는 핵심 메시지
      - 주제가 플롯을 통해 어떻게 드러나는지

   각 이벤트에 예상 회차 번호와 관련 캐릭터를 포함해주세요.
   ")
   ```

3. **파일 생성**
   - `plot/main-arc.json`
