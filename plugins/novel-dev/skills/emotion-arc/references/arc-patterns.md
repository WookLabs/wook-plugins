# 6가지 기본 감정 아크 상세 레퍼런스

MIT 컴퓨터 과학 연구소의 Andrew Reagan 등이 Project Gutenberg의 1,700여 편의 영어 소설을 감정 분석하여 도출한 6가지 기본 서사 아크입니다.

> **출처:** Reagan, A.J., Mitchell, L., Kiley, D., Danforth, C.M., & Dodds, P.S. (2016).
> "The emotional arcs of stories are dominated by six basic shapes."
> *EPJ Data Science*, 5(1), 31.

---

## 1. Rags to Riches (상승)

**패턴:** ↗ 지속 상승
**한국어:** 누더기에서 왕자로 / 역전 성공담
**감정 곡선:** 낮은 출발 → 꾸준한 상승 → 높은 마무리

```
감정 +10 |                    *
      +5 |              *  /
       0 |        *  /
      -5 |   *  /
     -10 | *
         +--------------------
          1  2  3  4  5  (act)
```

### 특징
- 주인공이 역경에서 시작하여 점진적으로 성장/성공
- 독자에게 희망과 성취감을 줌
- 가장 보편적으로 사랑받는 아크 유형

### 대표작
| 작품 | 장르 | 포인트 |
|------|------|--------|
| 신데렐라 | 동화 | 하녀 → 공주 |
| 알라딘 | 동화 | 거리의 소년 → 왕자 |
| 록키 | 영화/스포츠 | 무명 복서 → 챔피언 |
| 슬럼독 밀리어네어 | 영화/드라마 | 빈민가 → 퀴즈쇼 우승 |
| 대부분의 회귀/현판 웹소설 | 웹소설 | 약자 → 최강자 |

### 성공 조건
1. **설득력 있는 출발점**: 주인공의 결핍/불우함이 공감 가능해야 함
2. **점진적 성장**: 급격한 성장은 설득력 저하 (너무 쉬우면 안 됨)
3. **적절한 좌절**: 완전한 직선 상승은 지루함 → 소규모 하락 포함 권장
4. **성취의 대가**: 무엇을 희생하거나 배웠는지 명확해야 함
5. **카타르시스 마무리**: 끝이 충분히 보상적이어야 함

### 실패 패턴
- **금수저 주인공**: 출발점이 높으면 상승의 감동 없음
- **치트키 성장**: 노력 없는 성장은 독자 반감
- **단조로운 상승**: 갈등 없이 올라가기만 하면 긴장감 제로
- **미약한 결말**: 상승 끝에 보상이 빈약하면 허탈

### 변주
- **계단식 상승**: 상승 → 정체 → 상승 → 정체 (무협/성장물에 적합)
- **가속 상승**: 후반부로 갈수록 성장 속도 증가 (클라이맥스 효과)
- **지연된 상승**: 전반부 오래 정체 후 폭발적 성장 (반전형)

## 2. Riches to Rags (하강)

**패턴:** ↘ 지속 하강
**한국어:** 몰락 / 비극
**감정 곡선:** 높은 출발 → 꾸준한 하강 → 낮은 마무리

```
감정 +10 | *
      +5 |   *  \
       0 |        *  \
      -5 |              *  \
     -10 |                    *
         +--------------------
          1  2  3  4  5  (act)
```

### 특징
- 주인공이 좋은 위치에서 시작하여 점진적으로 몰락
- 비극적이지만 깊은 감정적 울림
- 독자에게 연민, 카타르시스, 때로는 경각심

### 대표작
| 작품 | 장르 | 포인트 |
|------|------|--------|
| 로미오와 줄리엣 | 비극 | 사랑의 시작 → 이중 자살 |
| 맥베스 | 비극 | 영웅 → 폭군 → 멸망 |
| 안나 카레니나 | 문학 | 상류층 → 사회적 파멸 |
| 브레이킹 배드 | 드라마 | 교사 → 마약왕 → 파멸 |
| 몰락형 다크히어로물 | 웹소설 | 강자 → 배신 → 파멸 |

### 성공 조건
1. **공감 가능한 시작**: 독자가 주인공의 전성기에 감정 이입해야 함
2. **필연적 하강**: 몰락의 원인이 캐릭터 결함에 뿌리 두어야 함 (하마르티아)
3. **저항의 순간**: 주인공이 운명에 저항하는 장면이 있어야 감동 배가
4. **의미 있는 결말**: 단순한 불행이 아닌 교훈/성찰 포함
5. **서브텍스트**: 표면적 하강 속에 숨겨진 의미 (성장, 깨달음 등)

### 실패 패턴
- **인과 없는 불행**: 이유 없이 나쁜 일만 연속되면 독자 이탈
- **자업자득 과다**: 주인공이 너무 밉상이면 공감 불가
- **단조로운 하강**: 변화 없이 떨어지기만 하면 지루
- **허무한 결말**: 아무 의미 없는 나쁜 결말은 불만족

### 변주
- **존엄한 몰락**: 하강하면서도 인간적 품위 유지 (비극의 미학)
- **느린 붕괴**: 서서히 무너지는 것을 독자만 알아챔 (아이러니)
- **반전 하강**: 겉은 성공인데 내면은 파괴 (이중 구조)

## 3. Man in a Hole (하강-상승)

**패턴:** ↘↗ 하강 후 상승
**한국어:** 구덩이에 빠진 사람 / 위기-극복
**감정 곡선:** 중간 출발 → 하강(위기) → 상승(극복) → 높은 마무리

```
감정 +10 |                    *
      +5 | *              /
       0 |   \        /
      -5 |     \  /
     -10 |       *
         +--------------------
          1  2  3  4  5  (act)
```

### 특징
- 가장 보편적이고 만족스러운 아크 유형
- "위기 → 극복"이라는 인간 보편 서사
- 독자가 주인공과 함께 고통받고, 함께 이겨내는 경험

### 대표작
| 작품 | 장르 | 포인트 |
|------|------|--------|
| 반지의 제왕 | 판타지 | 일상 → 모르도르 → 귀환 |
| 숀생크 탈출 | 드라마 | 억울한 수감 → 탈출 |
| 다이하드 | 액션 | 평범한 경찰 → 인질극 → 승리 |
| 대부분의 스릴러/추리 | 장르소설 | 사건 발생 → 위기 → 해결 |
| 회귀물 초반부 | 웹소설 | 죽음/실패 → 각성 → 반격 시작 |

### 성공 조건
1. **충분한 깊이의 구덩이**: 위기가 충분히 심각해야 극복의 감동이 큼
2. **유기적 해결**: 주인공의 노력/성장으로 탈출 (데우스 엑스 마키나 금지)
3. **바닥의 순간**: "All is Lost" 포인트가 명확해야 함
4. **상승의 가속**: 바닥을 찍은 후 상승은 빠를수록 쾌감
5. **변화된 귀환**: 구덩이 전/후 주인공이 달라져야 함

### 실패 패턴
- **얕은 구덩이**: 위기가 약하면 극복도 감동 없음
- **운에 의한 탈출**: 우연히 해결되면 카타르시스 없음
- **과도한 구덩이**: 너무 길면 독자 지치고 이탈
- **원상복귀**: 극복 후 아무것도 변하지 않으면 의미 없음

### 변주
- **이중 구덩이**: 첫 번째 탈출 후 더 깊은 구덩이 (서스펜스 극대화)
- **점진적 탈출**: 바닥에서 조금씩 올라오며 여러 단계 극복
- **협력 탈출**: 동료와 함께 빠져나옴 (우정/팀워크 강조)

## 4. Icarus (상승-하강)

**패턴:** ↗↘ 상승 후 하강
**한국어:** 이카로스 / 추락하는 영웅
**감정 곡선:** 낮은 출발 → 상승(전성기) → 하강(추락) → 낮은 마무리

```
감정 +10 |       *
      +5 |     /   \
       0 |   /       \
      -5 | /           \
     -10 | *              *
         +--------------------
          1  2  3  4  5  (act)
```

### 특징
- 교만(hubris)에 대한 경고 서사
- 전성기의 짜릿함과 추락의 아이러니가 공존
- 독자에게 강렬한 인상과 경각심

### 대표작
| 작품 | 장르 | 포인트 |
|------|------|--------|
| 그레이트 개츠비 | 문학 | 아메리칸 드림 → 환멸 |
| 오즈의 마법사 | 동화 | 마법의 세계 → 현실 귀환 |
| 늑대의 거리 | 영화 | 월스트리트 성공 → 몰락 |
| 타노스 (인피니티 워) | SF | 승리 목전 → 패배 |
| 다크히어로 타락물 | 웹소설 | 각성/성장 → 대가 치름 |

### 성공 조건
1. **매력적인 상승기**: 독자가 성공을 함께 즐길 수 있어야 함
2. **씨앗 심기**: 상승기에 이미 하강의 단서를 뿌려야 함 (복선)
3. **터닝 포인트**: 상승→하강 전환이 필연적이면서 놀라워야 함
4. **하강의 속도감**: 상승보다 빠른 추락이 극적 효과 배가
5. **잔여감**: 끝난 후에도 여운이 남아야 함

### 실패 패턴
- **무의미한 상승**: 상승기가 결말과 무관하면 시간 낭비 느낌
- **예측 가능한 추락**: 독자가 너무 일찍 결말을 예측하면 지루
- **동기 부재**: 왜 추락했는지 불분명하면 불만족
- **감정 단절**: 상승기에 공감 못 하면 추락도 무감정

### 변주
- **자각한 이카로스**: 추락을 알면서도 선택하는 비장미
- **부분 착지**: 완전히 추락하지 않고 중간에 멈춤 (열린 결말)
- **대리 이카로스**: 주인공이 아닌 조력자가 이카로스 아크

## 5. Cinderella (상승-하강-상승)

**패턴:** ↗↘↗ 상승-하강-상승
**한국어:** 신데렐라 / 시련을 넘은 해피엔딩
**감정 곡선:** 낮은 출발 → 상승 → 중간 위기 → 최종 상승 → 높은 마무리

```
감정 +10 |                    *
      +5 |    *          /
       0 |  /   \    /
      -5 |        \/
     -10 | *
         +--------------------
          1  2  3  4  5  (act)
```

### 특징
- 가장 인기 있는 아크 유형 (독자 선호도 1위)
- "잘 되다가 위기, 그리고 궁극적 승리"
- 로맨스, 판타지, 성장물의 기본 뼈대

### 대표작
| 작품 | 장르 | 포인트 |
|------|------|--------|
| 해리포터 시리즈 | 판타지 | 고아 → 마법사 → 위기 → 최종 승리 |
| 미녀와 야수 | 로맨스 | 만남 → 사랑 → 위기 → 해피엔딩 |
| 스타워즈 (에피소드 4-6) | SF | 농부 → 영웅 → 위기 → 승리 |
| 대부분의 로맨스 소설 | 로맨스 | 만남 → 설렘 → 오해/이별 → 재회 |
| 로판/판타지 웹소설 다수 | 웹소설 | 빙의/회귀 → 적응 → 위기 → 해피엔딩 |

### 성공 조건
1. **첫 상승의 매력**: 독자가 "잘 되길" 바라게 만들어야 함
2. **위기의 충격**: 중간 위기가 충분히 아파야 함 (가짜 위기 금지)
3. **바닥의 진정성**: 독자가 "정말 안 되는 건가?" 느껴야 함
4. **상승의 필연성**: 최종 승리가 운이 아닌 성장의 결과여야 함
5. **정서적 보상**: 첫 상승보다 최종 상승이 더 높아야 함

### 실패 패턴
- **가짜 위기**: 중간 하강이 너무 약하면 긴장감 없음
- **반복 감각**: 상승-하강이 단순 반복이면 패턴 노출
- **데우스 엑스 마키나**: 최종 상승이 우연에 의존하면 불만족
- **감정 과부하**: 등락이 너무 극단적이면 피로감

### 변주
- **다중 시련**: 2-3번의 하강을 거쳐 최종 상승 (장편에 적합)
- **팀 신데렐라**: 개인이 아닌 그룹 전체의 아크
- **내면 신데렐라**: 외적 성공보다 내면 성장이 핵심

## 6. Oedipus (하강-상승-하강)

**패턴:** ↘↗↘ 하강-상승-하강
**한국어:** 오이디푸스 / 거짓 희망의 비극
**감정 곡선:** 높은 출발 → 하강 → 잠깐 상승(거짓 희망) → 최종 하강

```
감정 +10 | *
      +5 |   \        *
       0 |     \    /   \
      -5 |       \/       \
     -10 |                  *
         +--------------------
          1  2  3  4  5  (act)
```

### 특징
- 가장 비극적인 아크 유형
- 중간의 "희망"이 마지막 추락을 더욱 고통스럽게 만듦
- 문학적 깊이와 복잡성이 가장 높음

### 대표작
| 작품 | 장르 | 포인트 |
|------|------|--------|
| 오이디푸스 왕 | 비극 | 왕 → 진실 추구 → 잠깐 희망 → 파멸 |
| 파우스트 | 문학 | 학자 → 악마 계약 → 쾌락 → 파멸 |
| 햄릿 | 비극 | 왕자 → 의심 → 확신 → 파멸 |
| 레퀴엠 포 어 드림 | 영화 | 꿈 → 희망 → 중독 → 파멸 |
| 비극적 복수물 | 웹소설 | 복수 시작 → 성공 눈앞 → 대가 |

### 성공 조건
1. **희망의 설득력**: 중간 상승이 진짜 가능성으로 느껴져야 함
2. **아이러니**: 희망이 오히려 파멸의 원인이 되는 구조
3. **불가피성**: 최종 하강이 "이렇게 될 수밖에 없었다" 느낌
4. **감정적 깊이**: 독자가 비극에서 카타르시스를 느낄 수 있어야 함
5. **의미**: 단순한 불행이 아닌 인간 조건에 대한 성찰

### 실패 패턴
- **거짓 희망의 빈약함**: 중간 상승이 너무 짧으면 효과 없음
- **과잉 비극**: 불행을 위한 불행은 독자를 지치게 함
- **인과 부재**: 왜 두 번째 하강이 발생하는지 불분명
- **카타르시스 부재**: 끝까지 보고 아무 감정이 없으면 실패

### 변주
- **깨달음의 추락**: 최종 하강에서 주인공이 진실을 깨달음
- **외면적 추락 + 내면적 상승**: 겉은 비극이나 영혼은 구원
- **세대적 비극**: 한 세대의 비극이 다음 세대의 교훈

### 장르별 주의사항

**로맨스:**
- Riches to Rags, Oedipus는 장르 위반 (해피엔딩 기대)
- Icarus 사용 시 반드시 에필로그에서 반전 필요
- 중간 위기(하강)는 반드시 관계 문제에서 발생해야 함

**스릴러:**
- 초반 긴장도가 너무 낮으면 안 됨 (첫 장면부터 긴장)
- 클라이맥스 전 반드시 "최악의 순간" 필요
- Cinderella 아크 사용 시 중간 상승이 "거짓 안전" 역할

**판타지:**
- 세계관 확장과 감정 아크가 동기화되어야 함
- 모험 출발 시점에서 첫 하강 또는 상승이 시작
- 귀환 테마 활용 시 Man in a Hole이 자연스러움

**웹소설 (전체):**
- 1화에서 감정 변화가 반드시 있어야 함 (독자 유치)
- 5화 이내에 첫 주요 상승 또는 카타르시스 필요
- 감정 평탄 구간이 3화 이상 지속되면 이탈 급증

## 아크 전환 기법

하나의 아크에서 다른 아크로 자연스럽게 전환하는 기법입니다.

### 전환 매트릭스

| 현재 아크 → 목표 아크 | 전환 기법 | 난이도 |
|----------------------|-----------|--------|
| Rags to Riches → Icarus | 성공이 교만을 부름 | 쉬움 |
| Rags to Riches → Cinderella | 성공 후 시련 추가 | 쉬움 |
| Riches to Rags → Man in a Hole | 바닥에서 반격 시작 | 보통 |
| Riches to Rags → Oedipus | 거짓 회복 삽입 | 보통 |
| Man in a Hole → Cinderella | 탈출 후 두 번째 위기 | 쉬움 |
| Man in a Hole → Oedipus | 탈출이 거짓 희망으로 전환 | 어려움 |
| Icarus → Oedipus | 추락 후 거짓 회복 삽입 | 보통 |
| Icarus → Man in a Hole | 추락을 바닥으로 재해석 → 반격 | 보통 |
| Cinderella → Oedipus | 최종 상승이 거짓으로 밝혀짐 | 어려움 |
| Oedipus → Cinderella | 최종 하강에서 진짜 반전 삽입 | 어려움 |

### 전환 시 핵심 원칙

1. **최소 3챕터 전환 구간**: 급격한 전환은 독자 혼란 유발
2. **동기 부여**: 전환의 원인이 캐릭터/플롯에 내재해야 함
3. **복선 필수**: 전환 전에 최소 2개 이상의 단서를 뿌려야 함
4. **독자 기대 관리**: 장르 약속을 위반하지 않는 범위에서 전환
5. **서브플롯 활용**: 메인 아크 전환 시 서브플롯으로 완충

### 전환 예시

**Rags to Riches → Cinderella (로맨스):**
```
챕터 1-4: 만남, 호감 상승 (Rags to Riches)
챕터 5: 오해/비밀 폭로 → 관계 위기 (전환 시작)
챕터 6-7: 이별/갈등 (하강)
챕터 8-10: 재회, 진심 확인 → 해피엔딩 (Cinderella 완성)
```

**Man in a Hole → Oedipus (스릴러):**
```
챕터 1-3: 사건 발생, 위기 진입 (하강)
챕터 4-6: 단서 발견, 희망 (상승 → Man in a Hole 예상)
챕터 7: 반전! 해결이 더 큰 위기를 촉발 (전환 시작)
챕터 8-10: 진짜 진실이 파멸적 (최종 하강 → Oedipus)
```

## 복합 아크 (장편/시리즈용)

장편 소설이나 시리즈에서는 단일 아크보다 복합 아크가 자주 사용됩니다.

### 매크로 아크 + 마이크로 아크

```
시리즈 전체 (매크로): Cinderella (↗↘↗)
  ├─ 1권: Man in a Hole (↘↗) - 주인공 첫 위기와 극복
  ├─ 2권: Rags to Riches (↗) - 성장과 성공
  ├─ 3권: Icarus (↗↘) - 교만과 추락
  ├─ 4권: Oedipus (↘↗↘) - 거짓 희망과 최악의 위기
  └─ 5권: Rags to Riches (↗) - 최종 승리
```

### 멀티 캐릭터 아크

서로 다른 캐릭터가 다른 아크를 따를 때 대비 효과:

```
주인공: Cinderella (↗↘↗) - 시련 후 해피엔딩
안타고니스트: Icarus (↗↘) - 성공 후 추락
조력자: Man in a Hole (↘↗) - 위기 극복
비극적 동료: Oedipus (↘↗↘) - 거짓 희망의 비극
```

### 서브플롯 아크 조합

메인 플롯과 서브플롯의 아크가 다를 때 감정 복합성 증가:

| 메인 아크 | 서브플롯 아크 | 효과 |
|-----------|-------------|------|
| Cinderella | Man in a Hole | 메인 하강 시 서브가 완충 |
| Man in a Hole | Rags to Riches | 바닥에서 서브가 희망 제공 |
| Icarus | Cinderella | 메인 추락 시 서브가 대조 |
| Oedipus | Rags to Riches | 비극 속 한 줄기 빛 |
