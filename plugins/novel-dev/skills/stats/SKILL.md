# /stats - 프로젝트 통계

현재 소설 프로젝트의 진행 상황을 표시합니다.

## 표시 정보
- 총 챕터 수 / 완료된 챕터
- 총 글자 수
- 막별 진행률
- 캐릭터 등장 횟수
- 복선 회수 현황
