# /design-timeline - 타임라인 설계

작품 내 시간 흐름과 이벤트를 설계합니다.

## Quick Start

```bash
/design-timeline              # 타임라인 설계
/design-timeline --regression # 회귀물 모드
```

## What It Creates

### plot/timeline.json
- 시간대 구분 (현재/과거/미래)
- 주요 이벤트 시간순 배치
- 시간 도약 포인트
- 회상 씬 배치
- 회귀물: 원래 타임라인 vs 변경된 타임라인

## When to Use

- 회귀물/시간여행 장르
- 과거 회상이 많은 스토리
- 여러 시점 교차 구조
