# Analyze Skill - Example Usage

## Example 1: "뭔가 이상한데" (일반적 불편함)

### 사용자 요청
```
사용자: 뭔가 이상한데, 뭐가 문제인지 모르겠어
```

### 스킬 응답
```
무엇이 이상하게 느껴지나요?

1. 전체적인 흐름
2. 특정 캐릭터
3. 특정 회차
4. 플롯 구조
5. 잘 모르겠음 (전체 스캔)
```

### 사용자 선택
```
사용자: 5번, 전체 스캔해줘
```

### 스킬 실행
```markdown
# 프로젝트 전체 분석 결과

전체 스캔을 수행한 결과, **페이싱 문제**가 발견되었습니다.

## 발견된 주요 문제

### 1. 챕터 12-18 구간 이탈 위험 (위험도: 85%)

**문제**:
- 행동 없이 내적 독백과 회상만 7회차 연속
- 플롯이 실질적으로 진행되지 않음
- 새로운 정보나 사건이 없음

**영향**:
- 독자가 "지루하다"고 느낄 가능성 매우 높음
- 예상 이탈률: 45%

**근거**:
```
chapters/chapter_012.json - 3개 씬 모두 회상
chapters/chapter_013.json - 2개 씬 모두 내적 독백
chapters/chapter_014.json - 1개 씬만 대화, 2개 씬 회상
...
```

**권장 조치** (우선순위: 높음):
1. 챕터 13에 외부 갈등 추가
   - 예: 라이벌 등장, 사건 발생, 충격적 정보 전달
   - 위치: `chapters/chapter_013.json:scenes` - 새 씬 추가

2. 회상 씬 50% 축소
   - 챕터 12, 14, 15의 회상을 대화나 행동으로 전환

3. 복선 떡밥 추가
   - 챕터 14-16 사이에 `hook_003` 심기
   - 독자 호기심 유지

---

### 2. 캐릭터 "유나" 대화 목소리 평범함 (점수: 58/100)

**문제**:
- 유나의 대사를 다른 캐릭터가 말해도 위화감 없음
- 개성 있는 말투/어휘 선택 부재

**예시**:
`manuscripts/chapter_003.md:45`
```
유나: "그래, 알겠어. 그렇게 하자."
```

이 대사는 준혁, 민지, 서연 누가 해도 어색하지 않습니다.

**권장 조치** (우선순위: 중간):
1. 유나만의 말버릇 정립
   - 제안: "음...", "그러니까...", 질문형 종결 자주 사용
   - 적용 범위: 챕터 1-20 대사 전면 수정

2. 어휘 특성 부여
   - 유나는 회사원 → 비즈니스 용어 섞어 쓰기
   - "ROI", "퍼포먼스", "피드백" 같은 단어 자연스럽게 사용

---

## 기타 발견사항

### ✅ 잘하고 있는 점
1. 플롯 구조는 탄탄함 (3막 구조 준수)
2. 세계관 일관성 양호
3. 복선 시스템 잘 설계됨 (10개 중 8개 회수 계획 명확)

### ⚠️ 개선하면 좋은 점
1. 서브플롯 "민지의 로맨스"가 메인플롯과 연결 약함
2. 챕터 엔드 훅이 약한 회차 3개 (ch 7, 12, 19)

---

## 다음 단계

1. [필수] 챕터 12-18 페이싱 개선
2. [권장] 유나 대화 목소리 차별화
3. [선택] 서브플롯 강화

수정 후 다시 분석하시려면:
`/analyze --focus=pacing --target=chapters/chapter_012-018.json`
```

## Example 2: 캐릭터 분석

### 사용자 요청
```
사용자: 유나 캐릭터 분석해줘. 독자들이 매력 없다고 하는데 왜 그런지 모르겠어.
```

### 스킬 실행
```markdown
# 캐릭터 "유나" (char_001) 분석 결과

## 종합 평가: 68/100 (개선 필요)

| 항목 | 점수 | 평가 |
|------|------|------|
| Want/Need 명확성 | 85/100 | ✅ 우수 |
| 내적 갈등 설득력 | 60/100 | ⚠️ 약함 |
| 캐릭터 아크 | 55/100 | ❌ 정체 |
| 대화 목소리 독특성 | 50/100 | ❌ 평범 |
| 행동 일관성 | 90/100 | ✅ 우수 |
| 독자 공감성 | 70/100 | ⚠️ 보통 |

---

## 주요 발견사항

### ✅ 강점

#### 1. Want vs Need가 명확함
**설정**: `characters/char_001.json`
```json
{
  "want": "회사에서 승진하여 인정받기",
  "need": "타인의 인정이 아닌 자기 자신의 가치 발견"
}
```

**평가**: 외적 목표(Want)와 내적 필요(Need)의 충돌 구조가 훌륭합니다.

**활용**: `chapters/chapter_001.json:scene_1`
- 유나가 승진을 위해 자존심을 굽히는 장면에서 갈등이 드러남

#### 2. 행동 패턴이 일관적
- 스트레스 → 술 마시기 (ch 1, 5, 12, 18)
- 갈등 회피 성향 (ch 3, 7, 15)
- 혼자 고민하는 습관 (ch 2, 8, 14)

독자가 "유나답다"고 느낄 수 있는 패턴이 잘 확립되어 있습니다.

---

### ❌ 문제점

#### 1. 캐릭터 아크가 정체됨 (치명적)

**발견**:
챕터 1-15 동안 유나가 실질적으로 성장하지 않습니다.

**근거**:
- 챕터 3: "나는 인정받고 싶어" (내적 독백)
- 챕터 7: "나는 인정받고 싶어" (동일한 고민 반복)
- 챕터 11: "나는 인정받고 싶어" (또 반복)
- 챕터 15: "나는 인정받고 싶어" (변화 없음)

**영향**:
- 독자가 "유나가 답답하다", "발전이 없다"고 느낌
- 캐릭터에 대한 흥미 상실

**해결책** (우선순위: 최고):

각 회차마다 갈등의 레이어를 추가하세요:
- 챕터 3: "인정받고 싶어"
- 챕터 7: "인정받아도 행복할까?" (의심 시작)
- 챕터 11: "행복이 뭐지?" (질문 확대)
- 챕터 15: "내가 진짜 원하는 건..." (깨달음 시작)

#### 2. 대화 목소리가 다른 캐릭터와 구별 안 됨

**예시 1**: `manuscripts/chapter_003.md:45`
```
유나: "그래, 알겠어. 그렇게 하자."
```

**예시 2**: `manuscripts/chapter_007.md:102`
```
유나: "좋은 생각이야. 그렇게 해봐."
```

**문제**:
- 이 대사들은 준혁, 민지, 서연 누가 해도 자연스럽습니다
- 유나만의 목소리가 없습니다

**비교**: 준혁의 경우
```
준혁: "흠... 재밌네. 한번 해볼까?" (ch 2)
준혁: "그렇게 간단한 문제였어? 하." (ch 5)
```
→ 준혁은 냉소적, 간결한 말투가 일관적입니다.

**해결책** (우선순위: 높음):

유나만의 대화 패턴을 정립하세요:

1. **말버릇 추가**
   - "음...", "그러니까..." 자주 사용
   - 단정적 표현 피하고 질문형으로 끝내기
   - "~인 것 같아", "~지 않을까?" 같은 완곡 어법

2. **어휘 특성**
   - 회사원이므로 비즈니스 용어 자연스럽게 섞기
   - "퍼포먼스", "피드백", "ROI" 같은 단어

3. **문장 구조**
   - 짧고 끊어지는 문장 (망설임 표현)
   - "그게... 음... 나는 말이야..." 같은 패턴

**수정 예시**:
❌ Before: "그래, 알겠어. 그렇게 하자."
✅ After: "음... 그럼, 그렇게 해볼까? 괜찮을 것 같은데."

**적용 범위**: 챕터 1-20 전체 대사 수정

#### 3. 내적 갈등이 추상적

**예시**: `manuscripts/chapter_011.md:89`
```
유나는 복잡한 기분이었다. 무언가 잘못된 것 같았다.
```

**문제**:
- "복잡한 기분", "무언가 잘못된" → 너무 모호함
- 독자가 유나의 감정을 구체적으로 느낄 수 없음

**해결책**:

추상적 감정을 구체적 생각/신체 반응으로 전환:

❌ Before: "유나는 복잡한 기분이었다."
✅ After: "가슴이 답답했다. 승진을 원한다고 말했지만, 그게 정말 자신이 원하는 건지 확신이 서지 않았다."

## 캐릭터 비교 분석

### 유나 vs 준혁 (남주)

| 요소 | 유나 | 준혁 |
|------|------|------|
| 대화 목소리 | 50/100 (평범) | 85/100 (독특) |
| 내적 갈등 | 60/100 (추상적) | 80/100 (구체적) |
| 캐릭터 아크 | 55/100 (정체) | 75/100 (진행 중) |

**발견**:
- 준혁에 비해 유나가 덜 매력적으로 느껴지는 이유가 명확합니다
- 준혁은 독특한 목소리 + 명확한 성장이 있는 반면, 유나는 정체되어 있습니다

**권장**:
주인공(유나)이 조연(준혁)보다 덜 매력적이면 안 됩니다.
유나의 대화 목소리와 캐릭터 아크를 우선적으로 강화하세요.

## 독자 반응 예측

beta-reader 관점에서 유나에 대한 독자 반응 예측:

### 긍정적 반응 (30%)
- "공감된다", "나도 저런 고민 있어"
- 현실적인 캐릭터라는 평가

### 부정적 반응 (55%)
- "답답하다", "왜 안 변해?"
- "재미없다", "주인공이 매력 없음"
- 챕터 10 이후 유나에 대한 관심 하락 예상

### 중립적 반응 (15%)
- "그냥 그렇다", "나쁘진 않은데..."

**결론**: 현재 상태로는 과반의 독자가 불만을 느낄 가능성 높음

   - 소요: 8-10시간

### 🟡 우선순위 높음

3. **내적 갈등 구체화**
   - 대상: 챕터 11, 14, 18
   - 방법: 추상적 표현을 구체적 생각/신체 반응으로
   - 소요: 2-3시간

### 🟢 우선순위 중간

4. **외적 특징 추가**
   - 습관/버릇 추가 (머리카락 만지기, 펜 돌리기 등)
   - 소요: 1시간

## Example 3: 플롯 분석

### 사용자 요청
```
사용자: 플롯 구조 분석해줘. 긴장감이 없다는 피드백이 많아.
```

### 스킬 실행
```markdown
# 플롯 구조 분석 결과

## 종합 평가: 65/100 (개선 필요)

긴장감 부족의 원인을 2가지 발견했습니다:
1. 페이싱 문제 (정체 구간 다수)
2. 갈등 확대 실패 (플랫한 긴장도 곡선)

---

## 1. 구조 분석

### 3막 구조 비율

| 막 | 현재 비율 | 이상적 비율 | 평가 |
|----|-----------|-------------|------|
| Act 1 (Setup) | 20% (ch 1-10) | 25% | ⚠️ 약간 짧음 |
| Act 2 (Confrontation) | 60% (ch 11-40) | 50% | ❌ 너무 김 |
| Act 3 (Resolution) | 20% (ch 41-50) | 25% | ⚠️ 약간 짧음 |

**문제**:
- Act 2가 과도하게 길어 "늘어지는" 느낌
- Act 1이 짧아 캐릭터/세팅 확립 부족

**해결책**:
```
현재: [1-10] [11-40] [41-50]
수정: [1-12] [13-38] [39-50]
```

챕터 11-12를 Act 1으로, 39-40을 Act 3으로 재분류하세요.

---

### 주요 전환점 평가

#### Plot Point 1 (Act 1 → Act 2 전환)
**위치**: `chapters/chapter_010.json`
**내용**: 유나가 계약 연애에 동의함

**평가**: ✅ 강력함 (75/100)
- 명확한 세계 변화 (일상 → 가짜 연애)
- 되돌릴 수 없는 결정

#### Midpoint (Act 2 중간)
**위치**: `chapters/chapter_025.json`
**내용**: 유나가 준혁에게 진짜 감정을 느낌을 자각

**평가**: ⚠️ 약함 (45/100)
- 내적 변화만 있음 (외적 사건 부재)
- Midpoint는 "게임을 바꾸는 사건"이어야 하는데, 내적 깨달음만으로는 부족

**문제**:
Midpoint가 약하면 Act 2 후반이 힘을 잃습니다.

**해결책**:
챕터 25에 외적 사건 추가:
- 옵션 1: 비밀 폭로 (누군가 가짜 연애를 눈치챔)
- 옵션 2: 적대자 등장 (준혁의 전 여친 등장)
- 옵션 3: 계획 실패 (회사 사람들 앞에서 들통)

#### Plot Point 2 (Act 2 → Act 3 전환)
**위치**: `chapters/chapter_040.json`
**내용**: 유나가 준혁과 결별 결심

**평가**: ✅ 좋음 (70/100)
- Dark Night of the Soul 잘 표현
- Act 3 Climax로 자연스럽게 연결

#### Climax
**위치**: `chapters/chapter_047.json`
**내용**: 유나가 준혁에게 진심 고백

**평가**: ⚠️ 보통 (60/100)
- 감정적 절정은 있으나, 외적 장애물 부족
- 너무 쉽게 해결되는 느낌

**해결책**:
Climax에 외적 장애 추가:
- 준혁이 이미 떠난 상황 (공항 추격 클리셰지만 효과적)
- 또는 유나가 극복해야 할 최종 시험 (회사 vs 사랑 선택)

## 2. 긴장도 곡선 분석

이상적인 긴장도 곡선:
```
긴장도
 100|                          /\
    |                         /  \
    |                    /\  /    \
    |          /\   /\  /  \/      \___
  50|     /\  /  \ /  \/
    |    /  \/    \
    |___/
  0 +---------------------------------->
      1    10   20   30   40   50 (챕터)
      PP1       MP        PP2   Climax
```

현재 긴장도 곡선:
```
긴장도
 100|
    |          /\
    |     /\  /  \_______________  /\
  50|    /  \/                   \/  \___
