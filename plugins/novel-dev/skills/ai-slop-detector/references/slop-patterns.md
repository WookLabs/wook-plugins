# AI-Slop 패턴 사전

한국어 AI 생성 텍스트에서 빈출하는 패턴과 대체 표현 모음.
`ai-slop-detector` 스킬에서 참조합니다.

## 사용법

### 스킬에서의 참조

`ai-slop-detector` 스킬은 이 파일의 패턴 목록을 기반으로 원고를 스캔합니다.

### 심각도 기준

| 심각도 | 의미 | 조치 |
|--------|------|------|
| HIGH | AI 생성 강력 의심. 즉시 재작성 권장 | 자동 플래그 + 재작성 제안 |
| MEDIUM | AI 패턴이나 문맥에 따라 허용 가능 | 플래그 + 검토 권장 |
| LOW | 남용 시에만 문제. 빈도 기반 판단 | 회당 N회 초과 시 경고 |

### LOW 심각도 빈도 기준

| 표현 | 허용 빈도 (3000자 기준) |
|------|----------------------|
| 마침내 | 1회 이하 |
| 고개를 끄덕이다 | 2회 이하 |
| 한숨을 내쉬다 | 2회 이하 |
| 입술을 깨물다 | 1회 이하 |
| 시선을 돌리다 | 2회 이하 |
| ~한 기분이 들었다 | 2회 이하 |
| ~라는 생각이 들었다 | 2회 이하 |

### 스타일 가이드 예외

`meta/style-guide.json`에 다음이 정의된 경우 해당 표현을 허용 목록에 추가:

```json
{
  "allowed_expressions": ["심장이 요동치다"],
  "character_speech_patterns": {
    "char_001": ["실로", "그리하여"]
  }
}
```

캐릭터 고유 화법으로 등록된 표현은 해당 캐릭터의 대사/내적독백에서 감지 제외.
