### Hook-Based Workflow Control

- **Ralph Loop Persistence**: Stop hook prevents premature exit during write-all
- **Schema Validation**: PreToolUse hook validates JSON before writing
- **Session Initialization**: SessionStart hook displays project status

### Progressive Disclosure Structure

All major skills now follow a 3-tier documentation structure:
- `SKILL.md` - Quick reference (~500 words)
- `references/detailed-guide.md` - Technical deep-dive
- `examples/example-usage.md` - Practical workflows

### Parallel Verification System

New `/verify-chapter` command orchestrates 3 validators simultaneously:
- critic (quality assessment)
- beta-reader (engagement analysis)
- genre-validator (genre compliance)

With confidence-based filtering (â‰¥75) and structured JSON output.

### New Agents (v3.0+)
