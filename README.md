# WookLabs Claude Code Plugins

WookLabs에서 제공하는 Claude Code 플러그인 마켓플레이스입니다.

## 설치 방법

### 마켓플레이스 추가
```bash
/plugin marketplace add WookLabs/wook-plugins
```

### 개별 플러그인 설치
```bash
/plugin install novel-dev@wook-plugins
```

## 플러그인 목록

### novel-dev
AI 소설 창작을 위한 멀티에이전트 오케스트레이션 플러그인

- **버전**: 3.3.3
- **카테고리**: productivity
- **키워드**: novel, writing, creative, ai-writing, storytelling

자세한 내용은 [plugins/novel-dev/README.md](./plugins/novel-dev/README.md)를 참조하세요.
